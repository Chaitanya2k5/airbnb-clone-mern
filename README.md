# Airbnb Clone - MERN Stack Project

GitHub Repository:
https://github.com/Chaitanya2k5/airbnb-clone-mern

## Features
- User Registration & Login
- JWT Authentication
- Property Listings
- Search Properties by Location
- Property Details Page
- Add Property
- Edit Property
- Delete Property
- Booking System
- My Properties Dashboard
- My Bookings Dashboard
- MongoDB Atlas Integration

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Axios

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
npm install
npm run dev
```
